Arquitectura del Computador

La arquitectura del computador describe el diseño conceptual y la estructura operativa de un sistema de cómputo. Incluye el conjunto de instrucciones, diseño del hardware y la forma en que los componentes interactúan para ejecutar programas.

Componentes principales

  • CPU: ejecuta instrucciones.
  • Memoria: almacena datos y programas.
  • Entrada / Salida: permite interacción con dispositivos.
  • Buses: transportan datos entre componentes.

CPU

La CPU interpreta y ejecuta instrucciones de programas.

Funciones

  • Leer instrucciones
  • Decodificarlas
  • Ejecutar operaciones
  • Controlar transferencia de datos

Componentes

  • Unidad de Control
  • ALU
  • Registros
  • Caché

Tipos de Arquitectura

Von Neumann

  • Una sola memoria para datos e instrucciones
  • CPU, memoria e I/O conectados por buses

Harvard

  • Memoria de datos separada
  • Memoria de instrucciones separada

RISC

  • Conjunto reducido de instrucciones
  • Ejecución rápida

CISC

  • Conjunto complejo de instrucciones
  • Mayor número de operaciones por instrucción

EPIC

  • Ejecución paralela explícita

VLIW

  • Instrucciones largas que ejecutan múltiples operaciones

Tipos de procesadores

  • CPU de uso general
  • Microcontroladores
  • Procesadores de señales digitales
  • GPU

Arquitectura MIPS

MIPS significa Microprocessor without Interlocked Pipeline Stages.

Características

  • Arquitectura RISC
  • 32 registros
  • Instrucciones de 32 bits
  • Pipeline de 5 etapas

Pipeline

  1. Instruction Fetch
  2. Instruction Decode
  3. Execute
  4. Memory
  5. Write Back

Registros MIPS

Registro Uso
$zero Constante 0
$at Uso del ensamblador
$v0-$v1 Valores de retorno
$a0-$a3 Argumentos
$t0-$t9 Temporales
$s0-$s7 Variables salvadas
$sp Puntero de pila
$ra Dirección de retorno

Lenguaje Ensamblador

Lenguaje de bajo nivel que representa instrucciones de máquina mediante nemotécnicos.

Componentes

  • Etiquetas

Modos de direccionamiento

Tipos de instrucciones MIPS

Formato R

add $s1,$s2,$s3 sub $t1,$t2,$t3 and $s4,$s5,$s6

Formato I

addi $s0,$s0,10 lw $t0,4($s0) sw $t1,8($s0)

Formato J

j etiqueta jal funcion

Ejemplo de bucle

li $s3,0 li $s4,10 Loop: slt $t0,$s3,$s4 beq $t0,$zero,End addi $s3,$s3,1 j Loop

Syscall

Imprimir entero

Leer entero

li $v0,5 syscall
