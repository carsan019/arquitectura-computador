Arquitectura del Computador

La arquitectura del computador describe la organización conceptual y funcional de un sistema de cómputo. Define cómo interactúan hardware y software para ejecutar programas.

Componentes principales

  • CPU
  • Memoria
  • Dispositivos de Entrada/Salida
  • Buses

CPU

La CPU es el cerebro del computador encargado de ejecutar instrucciones.

Componentes

  • Unidad de Control
  • ALU (Unidad Aritmético Lógica)
  • Registros
  • Memoria Caché

Funciones

  • Leer instrucciones desde memoria
  • Decodificar instrucciones
  • Ejecutar operaciones
  • Controlar flujo de datos

Tipos de Arquitectura

Arquitectura Von Neumann

  • Datos e instrucciones comparten memoria
  • Un solo bus

Arquitectura Harvard

  • Memoria de datos separada
  • Memoria de instrucciones separada

RISC

  • Conjunto reducido de instrucciones
  • Mayor velocidad

CISC

  • Conjunto complejo de instrucciones
  • Más operaciones por instrucción

MIPS

MIPS significa Microprocessor without Interlocked Pipeline Stages.

Características

  • Arquitectura RISC
  • 32 registros
  • Instrucciones de 32 bits
  • Pipeline de 5 etapas

Pipeline

  1. Fetch
  2. Decode
  3. Execute
  4. Memory
  5. Write Back

Registros MIPS

Registro Uso
$zero Valor constante 0
$v0-$v1 Valores de retorno
$a0-$a3 Argumentos
$t0-$t9 Temporales
$s0-$s7 Variables guardadas
$sp Puntero de pila
$ra Dirección de retorno

Lenguaje Ensamblador

Lenguaje de programación de bajo nivel que utiliza nemotécnicos para representar instrucciones de máquina.

Componentes

  • Instrucciones
  • Operadores
  • Etiquetas

Modos de Direccionamiento

  • Directo → dirección del dato
  • Indirecto → dirección de una dirección
  • Registro → dato en registro

Tipos de Instrucciones MIPS

Tipo R

add $s1,$s2,$s3

Tipo I

addi $s0,$s0,10 lw $t0,4($s0) sw $t1,8($s0)

Tipo J

j etiqueta

Instrucciones comunes

Instrucción Función
add Suma
sub Resta
lw Cargar memoria
sw Guardar memoria
and AND lógico
or OR lógico
slt Set if less than

Syscall

Imprimir entero

li $v0,1 move $a0,$t0 syscall

Leer entero

li $v0,5 syscall
